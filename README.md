# Dashboard de Viagens Oficiais do Governo Federal

Projeto desenvolvido com Python, Pandas, Plotly e Streamlit.

## Objetivo

Analisar os gastos com viagens oficiais registradas no Portal da Transparência em 2025.

## Principais análises

* Gastos por órgão
* Gastos por cargo
* Destinos mais frequentes
* Evolução mensal
* Viagens sigilosas
* Qualidade dos dados
* Painel executivo

## Tecnologias

* Python
* Pandas
* Plotly
* Streamlit
* OpenPyXL
